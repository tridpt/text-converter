# Ảnh chụp màn hình / GIF demo

Thư mục này chứa ảnh minh họa cho README.

## Cách chụp

1. Chạy app: `python -m app` rồi mở http://127.0.0.1:8000
2. Chụp các màn hình gợi ý:
   - `screenshot-home.png` — giao diện chính (kéo-thả file, chọn định dạng, tùy chọn)
   - `screenshot-matrix.png` — trang `/matrix` (ma trận định dạng)
   - `demo.gif` — quay lại một lượt: kéo file → chọn đích → xem trước → tải về
3. Lưu file vào thư mục `docs/` này.
4. Bỏ comment các dòng ảnh trong `README.md` (mục "Ảnh chụp màn hình").

## Gợi ý công cụ

- Windows: công cụ Snipping Tool (ảnh tĩnh); ScreenToGif (GIF).
- Nén ảnh trước khi commit để repo gọn nhẹ.
